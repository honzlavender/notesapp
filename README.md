# Welcome to my Notes App

A simple note taking app built with React Native and Swift

## Getting Started

git clone to local terminal

### Prerequisites

- node
- watchman
```
brew install node
brew install watchman
```

### Installation

1. clone repo

2. install dependencies
  ```
  npm install
  ```
3. Build with Xcode
  ```
  npx react-native run-ios
  ```
